Skill Gap Detector

A full-stack AI pipeline that scrapes job listings, tags them with tech stacks, and helps you discover what skills you need to land your next role.

See How It Works

What is this?

This project was built as part of the kyouth FX Digital Skills Program. Over 3 weeks, we built a data engineering pipeline, an AI layer, and a full-stack chat application — all containerized with Docker.

You paste your resume or upload a PDF, and the AI compares your skills against real Malaysian tech job listings to tell you exactly what you're missing.

The AI Pipeline

Week 1
Data Pipeline

Scraped 83 real job listings from Jobstreet Malaysia in .mhtml format. Processed them through a Medallion Architecture (Bronze → Silver → Gold) into a clean SQLite database with quality checks and deduplication.

Python SQLite BeautifulSoup Pydantic uv
Week 2
AI Layer

Used Google Gemini to extract tech stacks from job descriptions and resume text. Built an MCP (Model Context Protocol) server to abstract database access. Detected skill gaps by comparing resume skills against job market demand.

Gemini API FastMCP LangChain asyncio Pydantic
Week 3
Full-Stack App

Built a containerized chat application with a FastAPI frontend and backend. Users can chat or upload a PDF resume to get skill gap analysis. Includes a Charts.js dashboard for job market visualization.

FastAPI Docker Bootstrap Charts.js pdf.js

How to Use

Quick Start with Docker
# 1. Clone the repo
git clone https://github.com/Eiqbal25/kyouth-data-ai.git
cd kyouth-data-ai

# 2. Add your Gemini API key
mkdir week_3/secrets
echo "your_api_key" > week_3/secrets/google_api_key.txt

# 3. Run
cd week_3
docker compose up --build

Then open http://localhost:8000 and start chatting.

Using the Chat
  1. Type your skills or paste your resume text into the chat box
  2. Or click the upload icon to upload a PDF resume
  3. Press Send — the AI extracts your skills and compares them against 83 real job listings
  4. Get your top missing skills and full gap list instantly
  5. Visit /dashboard to explore the job market data with charts
